# 最小 A2UI 目录

此文件夹包含一个最小 A2UI 组件目录（`minimal_catalog.json`），用作测试新渲染器实现的测试平台。

## 目的

基础 A2UI 目录是全面的，具有许多组件、函数和布局原语。从头开始构建支持整个目录的新渲染器可能令人望而生畏。最小目录将表面积减少到一组核心基础组件：

- **Text**：用于渲染文本字符串。
- **Row**：用于水平弹性布局。
- **Column**：用于垂直弹性布局。
- **Button**：用于基本交互和操作分派。
- **TextField**：用于双向数据绑定的用户输入。

通过首先以这个最小目录为目标，新的渲染器实现可以建立坚实的基础——涵盖布局算法、组件嵌套、数据绑定和事件处理——然后再扩展到完整的基础目录。

## 示例

`examples/` 目录包含 5 个布局消息（`server_to_client_list` 格式）的 JSON 数组，演示了仅使用此最小目录中定义的组件的各种 UI 场景。它们用作基本的集成测试。
