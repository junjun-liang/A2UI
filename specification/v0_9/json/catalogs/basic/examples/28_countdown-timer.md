# 28_countdown-timer.json 说明文档

## 文件概述

本文件展示了倒计时卡片，演示了三列等宽布局和时间数据的展示。

## UI 场景

展示活动倒计时，包含事件名称、天/时/分三列倒计时数字和目标日期。

## 使用的组件

| 组件 | 用途 |
|------|------|
| Card | 卡片容器 |
| Column, Row | 布局 |
| Text | 事件名、倒计时数值和标签、目标日期 |

## 消息流程

1. **createSurface** — 创建界面
2. **updateComponents** — 构建倒计时布局
3. **updateDataModel** — 填充倒计时数据
