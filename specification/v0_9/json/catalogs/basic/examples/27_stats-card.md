# 27_stats-card.json 说明文档

## 文件概述

本文件展示了统计指标卡片，演示了 Icon 的动态绑定和趋势展示。

## UI 场景

展示一个统计指标卡片，包含指标图标、指标名称、数值和趋势（趋势图标和趋势文本均通过数据绑定）。

## 使用的组件

| 组件 | 用途 |
|------|------|
| Card | 卡片容器 |
| Column, Row | 布局 |
| Icon | 指标图标和趋势图标，name 通过数据绑定 |
| Text | 指标名、数值、趋势文本 |

## 示例关键片段

```json
{
  "id": "trend-icon",
  "component": "Icon",
  "name": { "path": "/trendIcon" }
}
```

趋势图标通过数据绑定动态设置，可切换为上升/下降等不同图标。
